# 🌿 Plant Leaf Disease Detector

This project is a simple web application that detects **plant leaf diseases** (Early Blight, Late Blight, or Healthy) using a **trained machine learning model** served with **FastAPI** and a beautiful HTML frontend.

---

## 📂 Files

- `main.py` — FastAPI backend API
- `index.html` — Frontend web page
- `my_model.keras` — Trained ML model
- `plantcute.png` & `leaf.png` — Cute UI icons
- `requirements.txt` — Python dependencies
- `start_project.bat` — Windows one-click launcher
- `run_project.py` — Python launcher for any OS

---

## 🚀 How to run this project

1️⃣ Clone this repo:
```bash
git clone https://github.com/Avik112/plant-leaf-disease-detector.git
cd plant-leaf-disease-detector
  




(Optional) Create a virtual environment:

 python -m venv venv
 venv\Scripts\activate   # For Windows
 # OR
 source venv/bin/activate   # For macOS/Linux

 pip install -r requirements.txt



Run the project:

Windows (one-click):
 
   start_project.bat



Or run manually:

python run_project.py
